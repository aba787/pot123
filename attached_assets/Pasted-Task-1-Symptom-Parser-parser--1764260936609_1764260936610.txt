Task 1: تحسين نظام فهم الأعراض (Symptom Parser)

سوّ لي parser يعتمد على كلمات مفتاحية ويصنّف الأعراض لثلاث مستويات:

عادي → يعطي نصيحة بسيطة + يذكر دواء مناسب لو ينفع

يحتاج معلومات إضافية → يسأل سؤال واحد فقط (مو نموذج كامل)

طارئ → يحوّل للصيدلي فورًا

الحالات الطارئة لازم تتحول مباشرة:

ضيقة نفس

تورم بالوجه/الشفاه

طفح جلدي شديد

تشنجات

فقدان القدرة على الكلام أو الحركة

حرارة + طفل أقل من 3 شهور

قيء شديد مستمر

Task 2: تحسين اكتشاف الأدوية (Drug Name Detection)

لازم يفهم أسماء الأدوية بكل أنواعها:

فيفادول = باراسيتامول

بندول = باراسيتامول

أدول = باراسيتامول

بروفين = إيبوبروفين

بونستان = Mefenamic Acid

ويتعرف على اسم الدواء حتى لو كتب المستخدم "وين ألقى فيفادول؟" أو "أخذت بروفين".

Task 3: فصل بين سؤال "عرض" و"دواء"

الحين البوت يخلط.
خل المنطق واضح:

إذا الجملة فيها عرض فقط → يحلل عرض

إذا فيها اسم دواء فقط → يعطي معلومات الدواء

إذا فيها الاثنين → الأولوية للتحويل لو حالة خطيرة

إذا السؤال عام مثل "دواء للصداع" → يقترح أدوية بدون لخبطة

Task 4: بدائل الدواء (Alternative Mapping)

خله يجاوب بشكل مبسّط وواضح، بدون شرح طبي طويل.

مثال للباراسيتامول:
“ما لقيتي فيفادول؟ خذي أي شيء باراسيتامول نفس المفعول مثل أدول أو بندول.”

Task 5: جرعات الأطفال (Pediatric Dose Logic)

لازم يسوي:

إذا عمر تحت سنتين → تحويل إجباري

إذا الوزن موجود → يحسب الجرعة حسب mg/kg

إذا وزن غير موجود → يسأل: “كم الوزن؟”

يمنع إعطاء إيبوبروفين تحت 6 شهور

يمنع أي دواء للرضع أقل من 3 شهور ويحوّل مباشرة

Task 6: معالجة النص العامي (Slang Normalization)

ضيف قاموس للكلمات الشائعة:

يعورني = ألم

يوجعني = ألم

كحه = سعلة

يكح = سعلة

يلتهب = التهاب

يلوع = غثيان

حلقي يلعب = التهاب حلق

صدري يسكر = ضيق تنفس

Rasi y3wrni → headache

Task 7: Flow واضح للسؤال (Decision Tree)

رتّب المنطق كالتالي:

Check for Emergency Symptoms → تحويل

Check for Child Age / Weight Rules

Drug Detected? → Give correct info

Symptom Detected? → Suggest appropriate OTC

Unknown? → Ask سؤال واحد فقط

لا تبين رسائل “ما فهمت” بدون سبب.

Task 8: إحالات الصيدلي (Pharmacist Escalation)

لازم الإحالة تكون فقط في الحالات التالية:

عرض خطير

سؤال عن جرعة لطفل أقل من سنتين

سؤال عن تداخلات خطيرة

استفسار غير واضح حتى بعد سؤال واحد فقط

Task 9: إزالة تكرار الردود الثابتة

الحين البوت كل شوي يطلع نفس النص.
خليه تكون الردود ديناميكية حسب الحالة.
